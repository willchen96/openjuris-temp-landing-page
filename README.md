# OpenJuris Landing Page

A clean, modern landing page for OpenJuris - a legal research platform for everyone.

## Features

-   Responsive design that works on all devices
-   Smooth scrolling animations
-   Modern gradient backgrounds
-   Clean typography

## Structure

-   `index.html` - Main HTML structure
-   `styles.css` - All styling and responsive design
-   `script.js` - Interactive features and animations
-   `images/` - Placeholder for feature screenshots

## Images Needed

To complete the landing page, add these screenshots to the `images/` folder:

1. `feature-search.png` - Screenshot of the search interface
2. `feature-answers.png` - Screenshot showing authoritative answers
3. `feature-context.png` - Screenshot showing contextual view

## How to View

Simply open `index.html` in your web browser.

## Customization

-   Update LinkedIn URLs in the Team section
-   Replace placeholder images with actual screenshots
-   Adjust colors by modifying the CSS variables (look for #4a9fd8 for the brand blue)
